## Descrição:

Uma API que recebe formulário e os armazena num banco de dados, permitindo também umas busca por data de envio.

## Instalação:

```bash
$ git clone https://github.com/JV-Albuquerque-S/teste-precato.git
$ cd teste-precato/
$ npm i
```

## Uso:

- rode o servidor em mode DEV:

```bash
$ cd teste-precato/
$ npm run dev
```
